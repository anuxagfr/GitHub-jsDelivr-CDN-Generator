Finding Your Details

To generate a link, the tool needs three key pieces of information from a GitHub repository URL.

For example, in the URL https://github.com/facebook/react:

  • The username is facebook.
  • The repository name is react.

The branch is usually main or master. You can find and change it using the branch selector dropdown on the repository's main page, typically on the top-left.

How to Use

  1. Sign In: Click the 'Login' button in the navigation bar and authenticate using the pop-up modal with your Google or GitHub account.
  2. Enter Details: Once logged in, the main tool will appear. Fill in the GitHub username, repository name, and the specific branch you want to access.
  3. Browse Files: Click the 'Browse Files' button. A file explorer will load, showing all the files and folders in that specific branch of the repository.
  4. Generate Link: Navigate through the folders and click on any file. The tool will instantly generate a globally-accelerated jsDelivr CDN link and a ready-to-use HTML snippet for your project.

Features

Easy CDN Link Generation

Converts standard GitHub file URLs into high-performance, globally distributed jsDelivr CDN links suitable for production use.

Live HTML Snippets

Intelligently creates the correct HTML tag for you. It generates <script> tags for .js files and <link> tags for .css files.

Direct Downloads

Provides a direct download link for any file in the repository, which is useful for grabbing individual assets without cloning the entire project.

Responsive UI

Ensures a seamless and intuitive experience whether you are on a desktop, tablet, or mobile phone.

Automatic Dark Mode

The interface automatically syncs with your operating system's theme for comfortable viewing, day or night.

Private Repo Support

You can optionally provide a GitHub personal access token to browse files in private repositories you have access to.
